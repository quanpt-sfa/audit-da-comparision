# Auditor-regime heterogeneity for CFS proxy validation

## Purpose

This extension tests whether the criterion validity of abnormal-CFO scores differs between Big4 and non-Big4 audit regimes. Auditor identity is joined only after expected-CFO scores are constructed and is never used as an expected-CFO predictor.

The extension also evaluates within-firm dynamics around auditor-tier transitions. These switch analyses are deliberately separated from the pooled Big4/non-Big4 comparison because voluntary switching can be endogenous to distress, audit disagreements, reporting quality, or audit-fee pressure.

## TT200 time contract

The project distinguishes three related windows:

- source and target-construction period: fiscal years 2015-2025;
- rolling-training start: fiscal year 2015;
- theoretical out-of-sample test period: fiscal years 2016-2025.

Fiscal year 2015 is the first TT200 source and warm-up year. It is not treated as an out-of-sample model test year because no earlier TT200 observation exists for model training. The effective first test year may be later than 2016 when the prespecified minimum training-sample requirement is not satisfied.

The standard runner locks:

- `source_start_year = 2015`;
- `source_end_year = 2025`;
- `training_start_year = 2015`;
- `test_start_year = 2016`;
- `test_end_year = 2025`.

Auditor metadata retain 2015 so that consecutive-year switch histories can use it as a prespecified pre-test year. Big4/non-Big4 criterion-validity metrics and yearly-AUC tests use only common-primary model test observations from 2016 onward. Direct target event studies may use 2015 as pre-event support but never use pre-2015 information.

## Auditor normalization

Original audit-firm names are retained and normalized into:

- `BIG4`;
- `NON_BIG4`;
- `UNKNOWN`;
- `AMBIGUOUS`.

Big4 matching covers Deloitte, PwC/PricewaterhouseCoopers, EY/Ernst & Young, and KPMG. Missing auditor values are never coded as non-Big4.

## Pooled auditor-regime analysis

The criterion-validity analysis uses `earnings_working_capital` scores from:

- `common_primary_models`;
- `analysis_core`.

The pooled sample remains unchanged. Auditor coverage does not filter the main validation results. Cross-model and cross-auditor comparisons use the common issuer-year intersection generated by the prespecified models.

The focal pooled parameter is the interaction between the outcome-specific score and the Big4 indicator. A prevalence difference alone does not identify audit quality because Big4 and non-Big4 clients are selected populations. Pooled results therefore describe audit-regime-dependent criterion validity, not a causal effect of auditor choice.

## Direction-specific switch event study

Event time zero is the first fiscal year audited by the new auditor tier. Switches are classified separately as:

- `UPGRADE`: non-Big4 to Big4;
- `DOWNGRADE`: Big4 to non-Big4.

The primary clean-switch design requires two consecutive pre-event years in the old tier and event years 0, +1 and +2 in the new tier. Missing auditor years, reversals within the event window and later same-direction events for the same issuer are excluded from the primary event set.

For each treated event, controls must remain continuously in the treated firm's pre-event auditor tier throughout the full event window. Controls are exact-matched at event time -1 on exchange and industry. The stacked linear probability model absorbs:

- issuer-by-event-stack fixed effects;
- calendar-year-by-event-stack fixed effects.

Event time -1 is the omitted reference period. Standard errors are clustered by issuer. A joint Wald test evaluates all estimable pre-event leads. The estimand is a within-firm dynamic association around an auditor-tier transition, not an automatic causal effect.

The primary base-rate outcomes are constructed from the direct CFS target table rather than from model availability:

- any candidate;
- audited CFO decrease or increase;
- CFF-down and CFI-up candidates;
- signed and absolute CFO correction magnitudes.

This separation prevents expected-CFO model availability from defining the event-study outcome sample.

## Borrowing-intensity mechanism

When line-item data are available, pre-event borrowing intensity is computed from preliminary borrowing proceeds in event times -2 and -1, scaled by the configured positive lag-assets denominator. It is not measured from audited post-switch values.

High/low borrowing groups are defined within switch direction using the treated-event median. The mechanism analysis asks whether CFF-down dynamics are stronger among firms with greater pre-switch borrowing exposure and whether the same pattern is absent from the heterogeneous CFI-up channel.

## Reversible-treatment dynamic DiD

The dynamic DiD uses the same event-specific switcher and stable-stayer stacks. It estimates changes from event time -1 for horizons -2, 0, +1 and +2 separately for upgrades and downgrades.

Baseline overlap weights are estimated from available pre-event size, preliminary CFO, borrowing intensity, prior correction outcomes, cohort, exchange and industry. When minimum propensity-model support is not met, the pipeline records an explicit unweighted fallback rather than silently extrapolating.

Uncertainty is obtained by resampling treated issuers. Negative horizons are reported as placebo/pretrend contrasts. Because auditor tier can switch in both directions and voluntary switching remains endogenous, this module is labelled reversible dynamic DiD but its estimates retain an associational interpretation unless parallel trends, no anticipation and conditional comparability are credible.

## Formal temporal-heterogeneity tests

Year-specific AUC ranges are not compared naively with auditor-group differences. The pipeline instead produces two formal tests:

1. an issuer-cluster bootstrap of the full yearly-AUC vector, followed by a generalized Q statistic using the estimated covariance matrix;
2. a score-by-year logistic interaction model with issuer-clustered covariance and a joint Wald test.

The claim that temporal heterogeneity exceeds auditor heterogeneity is retained only when at least one formal test rejects homogeneous ranking validity and the supported yearly cells have adequate positive and negative counts.

## Interpretation

A discipline interpretation is strongest when downgrade and upgrade estimates have opposite signs in the borrowing-related CFF channel, survive stable-control and overlap-weighted analyses, pass pretrend checks, and are not mirrored in the heterogeneous CFI channel. If deterioration precedes a Big4 downgrade, the result is evidence of endogenous sorting or auditor-client realignment rather than a treatment effect of leaving Big4.
